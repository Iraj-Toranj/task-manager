\# 🧠 Task Manager (Python Tkinter)



A simple desktop task manager built with Python and Tkinter.



\## 🚀 Features

\- Add / Delete / Edit tasks

\- Mark tasks as completed

\- Priority system (High / Medium / Low)

\- Search tasks

\- Filter (All / Pending / Completed)

\- Progress bar



\## 🛠️ Tech Stack

\- Python

\- Tkinter



\## ▶️ How to Run

python main.py
